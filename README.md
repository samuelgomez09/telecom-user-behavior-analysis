# Análisis de Comportamiento de Usuarios y Segmentación Comercial | ConnectaTel LATAM

¡Bienvenido/a a este proyecto de analítica de datos aplicado al sector de telecomunicaciones! Este repositorio contiene el desarrollo de punta a punta de un análisis estratégico para **ConnectaTel**, una empresa de telecomunicaciones con operaciones en México y Colombia.

## 📄 Descripción del Proyecto
El objetivo principal es entender cómo los clientes consumen realmente sus servicios móviles (llamadas y mensajes) mediante la integración de múltiples fuentes de datos. El proyecto abarca desde la limpieza y validación de datos (detección de inconsistencias y valores atípicos con el método IQR) hasta la segmentación demográfica y de comportamiento para proponer mejoras en la oferta comercial de la compañía.

## 🎯 Preguntas de Negocio Resueltas
1. **¿Qué segmentos de clientes muestran mayor o menor uso de llamadas y mensajes?** (Segmentación por niveles de uso).
2. **¿Qué usuarios presentan valores atípicos (outliers)?** Identificación de consumos inusuales que alerten sobre posibles fraudes o errores de registro.
3. **¿Cómo varía el uso según la edad y el tipo de plan contratado?** Análisis del comportamiento intergeneracional en el mercado de LATAM.
4. **¿Qué patrones clave optimizan la oferta comercial?** Recomendaciones accionables para el diseño de planes de telefonía más atractivos.

## 🛠️ Tecnologías y Herramientas Utilizadas

* **Visualización de Datos:** Matplotlib, Seaborn
